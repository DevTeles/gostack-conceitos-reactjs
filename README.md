## Como baixar o projeto

```bash

  # Clonar o repositório
  $ git clone https://github.com/DevTeles/gostack-template-conceitos-reactjs.git

  # Entrar no diretório
  $ cd gostack-template-conceitos-reactjs

  # Instalar as dependências
  $ yarn

  # Iniciar o projeto
  $ yarn start
```
